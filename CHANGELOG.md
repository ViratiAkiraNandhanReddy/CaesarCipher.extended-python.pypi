## [1.0.0] - 2026-05-16

### Added
- Core Caesar cipher encryption and decryption functionality
- Customizable shift parameter for encryption/decryption
- Support for altering digits (`alterNumbers` / `isNumbersAltered` flags)
- Support for altering symbols (`alterSymbols` / `isSymbolsAltered` flags)
- File-based encryption via `encrypt_file()` method
- File-based decryption via `decrypt_file()` method
- Input validation for text, shift values, and file operations
- Intuitive API with `Encryption` and `Decryption` classes
- Basic emoji support for encryption and decryption
- Support for both uppercase and lowercase letters
- Comprehensive unit tests for encryption and decryption functionality
- File I/O tests for encrypt/decrypt file operations
- PyPI package distribution
- GitHub Actions CI/CD pipeline for automated testing
- Comprehensive documentation and usage examples
